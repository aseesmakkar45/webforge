# WebForge Hackathon: Reusable AI SaaS Template

This repository contains the highly reusable, AI SaaS Dashboard Template designed for the Web Forge Hackathon.

The complete, implementation-ready architectural blueprints can be found in the `template-1/docs/` directory:

## 🏗️ Project Structure
- `template-1/frontend/`: Next.js 15 application (UI, 3D WebGL, Animations).
- `template-1/backend/`: Node.js Express server (Auth, Gemini AI, MongoDB).
- `template-1/docs/`: Architectural documentation and blueprints.

## 🚀 Collaborative Git Workflow
If you are contributing to this project, please follow the established branch workflow:
1. Always base your work off the `main` branch.
2. Use the `frontend` branch for UI/Next.js tasks.
3. Use the `backend` branch for API/Express tasks.
4. Merge into `main` after completing milestones.
